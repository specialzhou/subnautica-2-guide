Current answer

Install EA 1.1 Hotfix 4, restart the game client, and temporarily disable any mods before loading the save. The official patch specifically fixes Angel Combs not opening and blocking progression. If it remains stuck after confirming the hotfix is installed, report the save through in-game feedback or Unknown Worlds support.
