Subnautica 2 是 Unknown Worlds 和 Krafton 联合开发的深海生存探索游戏，于 2026年5月14日 进入抢先体验（Early Access）。上市12小时内售出 200万份，Steam 历史峰值在线 467,582人，Twitch 同时观看峰值 413,200人，登上 Steam 历史并发排行第29位。

💡 对新手说一句话
游戏的"没有指引"设计是故意的，也是最好玩的部分。不要焦虑，也不要急着去深处——先把浅海玩透，装备自然就会解锁，进度自然会推进。

第一小时清单

从落地到站稳脚跟的完整路线图

很多新手开局不知道该做什么，这里给你一个分钟级的行动计划，照着做就能度过最危险的早期阶段。

  • 0–3分钟
    打开逃生舱里所有储物箱，拿走全部物品。特别注意：急救包、营养棒一定要拿，这是你最早的续命道具。
  • 3–5分钟
    按 Tab 打开 PDA，读完所有生存警报。游戏里没有任务栏，PDA 就是你的任务书。
  • 5–15分钟
    在逃生舱附近 0–30米 范围下水，捡起橙色岩石（钛矿石）和绿斑岩石（铜矿），各捡5块以上。这个深度很安全，不用担心氧气。
  • 15–20分钟
    用随身合成器合成：标准氧气罐（钛×1）。马上装备，潜水时间立刻延长。这是全游戏第一优先级合成。
  • 20–25分钟
    合成刀具（钛×1 + 硅胶×1）。硅胶来自逃生舱附近的橙色蘑菇状植物，用手直接采摘。刀可以采集植物，也能击退小鱼。
  • 25–35分钟
    在逃生舱外放置一个信标（Beacon）（铜×1 + 钛×1），防止你以后找不到回家的路。这一步很重要，很多新手第一次探索就迷路了。
  • 35–50分钟
    合成扫描仪（钛×1 + 电池×1），电池需要 铜×2 + 酸性蘑菇×2。开始扫描周围的生物和植物，扫描完成后 PDA 会解锁相关条目，有时会触发新图纸。
  • 50–60分钟
    找一块相对平坦的海底，放置你的第一个基础舱（Base Foundation）。别担心选址不完美——早期基地随时可以拆了重建，关键是现在就开始积累合成台。
💡 第一小时结束后你应该拥有
标准氧气罐、刀具、扫描仪、信标（×1个）、第一个基础舱雏形。如果这些都有了，你的开局是成功的。

1. 新手入门 — 游戏基础

开局做什么 · 界面说明 · 按键指南

刚开始要做的 5 件事

  1. 检查逃生舱内的所有储物箱，拿走急救包、营养棒和基础材料。
  2. 按 Tab 打开 PDA，阅读系统自动发送的生存警报，了解当前优先任务。
  3. 在舱外浅海（0–30 米）采集石英、钛矿石、铜矿，这是合成一切的基础。
  4. 合成氧气罐（需要钛×1），这是最优先的合成目标，马上扩充潜水时间。
  5. 在逃生舱周围放置一个信标，防止迷路找不到回家的路。

界面说明

💨

氧气条（蓝色）

屏幕左侧，在水面可自动回复。越深消耗越快，快没了赶紧上浮！

🍔

食物条

右侧橙色条。耗尽会减速和掉血，但不像氧气那么紧急，别慌。

💧

水分条

右侧蓝色条。喝水或净化海水补充，耗尽比食物更危险。

❤️

生命值（红色）

被生物咬或撞载具会减少，急救包回复。不冒险基本不会掉。

🌡️

温度警告

深层热泉区才出现，早期不用担心。出现时赶紧离开热泉区域。

📱

PDA（Tab键）

游戏的"任务书"，所有数据、图纸、故事线都在这里，多读！

2. 氧气与生存管理

潜水时间 · 食物饮水 · 不再溺死的秘诀

Subnautica 2 最容易死亡的原因是溺死，而不是被生物攻击。管好氧气，你就能活很久。

⚠️ 新手最常见死法
探索太投入忘看氧气条，等发现只剩一点点时已经来不及上浮。氧气条低于30%就要往上游，养成习惯。

氧气装备升级路线

装备所需材料增加潜水时间何时获取
🔵 标准氧气罐钛 ×1+30秒开局第一优先
🔷 高容量氧气罐钛 ×3 + 玻璃 ×1+90秒前30分钟合成
🔶 超高容量罐高容量罐 ×2+180秒中期主力装备
♻️ 再呼吸器铜 ×2 + 钛 ×1消耗速度减半中期辅助神器
💡 玻璃怎么合成？
玻璃 = 石英 ×2，在随身合成器就能做。石英是白色半透明晶体，在浅海沙地大量分布，很好认。

食物与饮水指南

早期（0–1小时）： 靠逃生舱里的营养棒撑一段时间。同时捡浅海里漂着的橙色浆果（直接食用），喝用过滤瓶净化的海水。

中期（1–5小时）： 在基地里建生物反应舱，种植外星菌菇。一旦建好，食物问题永久解决。

⚠️ 不要直接喝海水
外星星球海水盐度极高，直接喝会加速脱水！必须用过滤瓶（铜×1 + 钛×1 + 硅胶×1）净化后再喝。

3. 合成系统详解

配方 · 材料在哪里找 · 合成优先顺序

合成台介绍：三级体系

🎒

随身合成器

按 X 打开，随时随地可用。合成基础工具和早期装备，不需要基地。

🔧

工作台

需要放置在基地内。解锁中级图纸（海蛾、基础舱扩展等），是中期核心。

🏭

改装站

高级装备和载具改装在这里做。后期必备，用于制作深层装备。

材料在哪里找？完整指南

这是新手最常困惑的问题。以下每种材料都标注了外观、具体位置和采集难度：

材料外观特征在哪里找深度难度
🟠 钛矿石 橙棕色岩块，有金属光泽 浅海礁石底部、礁石表面大量分布，随处可见 0–100m 简单
🟢 铜矿石 绿色斑点岩，表面有孔洞 礁石区和沙地交界处，和钛矿石同一区域 0–150m 简单
⚪ 石英 白色半透明晶体，像玻璃碎片 浅海沙地上大量散布，很容易辨认 0–80m 简单
🍄 硅胶（橡胶） 不是矿石！是橙色蘑菇状植物的分泌物 用刀砍橙色海底蘑菇植物即可采集 0–60m 简单
🍋 酸性蘑菇 黄绿色发光小蘑菇，成片生长 礁石阴暗面和洞穴入口附近 0–100m 简单
⚡ 磁铁矿 黑色八面体晶体，会轻微发光 磁力异常区（指南针会乱转的地方），找异常区域 50–200m 中等

🟡 黄金 金黄色晶体，非常醒目 火山岩石区域（颜色偏红黑的地方） 200–500m 中等
💎 钻石 蓝色半透明宝石，发蓝光 深层熔岩带，必须有载具才能到达 500m+ 困难

4. 基地建设指南

选址 · 建造顺序 · 供电 · 结构完整性

基地是你在深海的永久立足点，也是联机玩法的核心枢纽。早期先建一个临时基地，等探索更多后再迁移到更好的位置。

💡 新手常见误区
不要等到"找到完美位置"才开始建基地。先随便搭一个有工作台的雏形，用了几小时再迁移也不晚。早建早用！

选址要点

🌿

靠近资源区

距主要矿脉不超过200米，减少来回时间。

☀️

阳光层（0–200m）

太阳能板可用，适合主要居住基地，供电简单。

🦈

远离掠食者巢穴

用扫描仪确认周边没有大型生物盘踞。

🪨

平坦海底

建造基础舱时需要相对平整的地面。

建造顺序（推荐）

  1. 放置基础舱（Base Foundation）作为起点——材料：钛×2 + 玻璃×1
  2. 放置走廊舱扩展空间，连接各功能舱
  3. 建造工作台——解锁中级合成，基地的核心
  4. 建造太阳能板（浅层）或生物反应堆（深层）供电
  5. 添加医疗舱和食物储存间，保障生存
  6. 建造水培室种植食物，实现自给自足
  7. 建造改装站，解锁高级装备图纸
  8. 最后建造潜艇坞，停放和充能海蛾等载具

供电系统说明

发电方式所需材料适用场景
☀️ 太阳能板石英×2 + 铜×1浅层基地（0–200m），白天自动充电，无消耗
🌿 生物反应堆钛×3 + 玻璃×2深层基地，用生物材料（鱼/植物）发电，持续稳定
🌊 热能发电机钛×5 + 磁铁矿×2热泉附近，发电量最大，适合深层永久基地
⚠️ 结构完整性
每个舱体有结构完整性（Hull Integrity）值。越深水压越高，需要加加固隔板（钛×3 + 玻璃×1，提供 +7 完整性）。每下潜100米消耗约1点完整性，完整性为0会进水！

5. 潜水艇与载具

海蛾 · 改装顺序 · 深层载具

没有载具，你永远无法到达中层以下的区域。海蛾一型是你的第一辆载具，也是游戏中期最常用的座驾。

载具最大深度特点适合阶段
🦋 海蛾一型200m（改装后900m）灵活小巧，速度快，单人为主早中期
🦾 PRAWN 外骨骼1700m适合岩石地形采矿，可以跳跃中后期
🐉 海龙（Sea Dragon）1700m+超大容量，可搭载海蛾，团队探索后期

海蛾一型：详细使用指南

海蛾是游戏里最早获得的载具，材料都在浅海，应该尽早合成。

⚡

电量管理

海蛾用电池驱动，快用完时及时回基地充电（潜艇坞会自动充）。

🔦

灯光

按 F 开关车灯，进入黑暗区域前一定要开灯，否则什么都看不见。

🐟

生物碰撞

被大型生物咬会伤到载具，修复工具可以修，但要小心血量。

📦

储物空间

默认4格，升级储物模块后扩至8格，出门前先清空手动背包。

海蛾改装优先级

  1. 深度模块 Mk.1：200m → 400m，获取中层矿物必须
  2. 储物扩展模块：存储格 ×2，采矿效率翻倍
  3. 鱼雷系统：遭遇大型掠食者的保命手段
  4. 深度模块 Mk.2：400m → 600m，进入蘑菇林区
  5. 深度模块 Mk.3：600m → 900m，解锁大部分中层区域

6. 四人联机技巧

分工方案 · 联机独有注意事项

Subnautica 2 首次引入最多 4人联机，是系列最大玩法革新。合理分工可以让资源积累速度翻4倍。

推荐分工方案

⛏️

采矿手 ×1

专注浅海资源采集，保证基地原料供给稳定，是队伍后勤核心。

🔬

科技官 ×1

驻守基地，负责合成装备、管理供电和储物，让其他人专心探索。

🗺️

探索者 ×1

驾驶海蛾深层侦察，扫描生物，寻找残骸图纸，绘制安全路线。

🛡️

护卫 ×1

保护采矿手和探索者，携带鱼雷和诱饵，应对大型掠食者。

联机注意事项

⚠️ 重要：共享储物
联机中所有人都能拿共享箱里的物品。提前约好谁管物资，否则关键材料会被误拿。建议科技官专门管理仓库。
💡 信标共享
任何人放置的信标对全队可见。建立统一命名规则，比如"矿点1""危险区域"，让全队都能看懂。
💬 推荐配合方式
联机时强烈建议开语音（Discord等）。游戏本身无语音功能，文字聊天在紧急情况下太慢。

8. 新手常见问题

最高频问题的直接解答
我一直在溺死，怎么办？+

溺死是新手最常见的死法。记住两条铁则：
① 氧气低于30%立刻往上浮，不要再贪心继续采集；
② 合成高容量氧气罐（钛×3 + 玻璃×1）比标准罐多3倍时间，一定要尽早升级。
另外，水面换气是免费的，多用"出水→换气→再下去"的节奏，不要一次潜太久。

游戏没有任务箭头，我该去哪里？+

Subnautica 2 的设计理念就是"自己探索"，没有任务箭头是故意的。正确的探索顺序：
① 读 PDA（Tab键）——所有故事线索都在里面，每次扫描残骸和生物都会更新；
② 跟着残骸走——水面和浅海会有大量坠落的飞船残骸，走近就会触发新内容；
③ 不要想太多，随便游就行——好东西都是"意外"发现的，享受探索本身。

被大型生物吓到不敢下水怎么办？+

这是正常反应！开发者专门设计了"深海恐惧感"。几个降低恐惧的实用技巧：
① 开灯——黑暗本身就是恐惧来源，随时开海蛾车灯；
② 带诱饵——声诱饵（磁铁矿×1 + 植物×1）能把大部分中型生物引开；
③ 了解生物——扫描仪扫过的生物会在 PDA 里显示习性，知道它吃什么才会攻击，恐惧感会大幅降低；
④ 大部分生物不会主动攻击——你先不碰它，它一般不咬你。

联机和单机哪个更适合新手？+

两种都可以，但各有优缺点：
单机： 体验更完整，自己掌控节奏，不用担心别人拿走你的资源，恐惧感更强烈（褒义）。
联机： 资源积累快4倍，有队友分工更安全，大型生物遇到时不那么怕，适合不喜欢孤独感的玩家。
建议：第一次玩先单机体验剧情，二周目再联机体验全新分工乐趣。

我的基地进水了怎么办？+

基地进水说明结构完整性降到0了。解决步骤：
① 先建造加固隔板（钛×3 + 玻璃×1）补强结构，每个提供 +7 完整性；
② 用修复工具在漏水点右键修复裂缝；
③ 如果大面积进水，先游出基地，加足够的加固隔板后再修复漏洞；
④ 基地深度每增加100米就消耗约1点完整性，注意提前准备加固隔板再建深层基地。

游戏里到了晚上怎么应对？+

Subnautica 2 有昼夜循环，夜晚水下一片漆黑，恐惧感翻倍。应对方法：
① 夜晚留在基地——整理物资、合成装备，等天亮再出发；
② 如果要出门，开全部灯光，随身携带荧光棒（丢出去照明）；
③ 夜晚有一些生物反而更安静，有经验的玩家会利用夜晚探索——但新手不建议。
④ 某些深层区域的生物在夜晚有不同的行为模式，扫描仪可以查到。

图纸怎么解锁？我合成不了很多东西。+

图纸解锁有三种方式：
① 扫描仪扫描残骸碎片——在水下找到飞船残骸的碎片，靠近后用扫描仪扫，扫满4片解锁一张图纸；
② 修复工具修复残骸电路板——靠近有损坏电路板的残骸，用修复工具修复，有时会给新图纸；
③ 读 PDA 里的数据记录——有些图纸通过故事事件自动解锁。
总结：多去找残骸，多扫描，多读 PDA，图纸自然越来越多。

9. 部署到 GitHub Pages

免费建站 · 三步上线 · 广告变现

将这个攻略页面上传到 GitHub，即可获得免费公开网址，便于分享和被搜索引擎收录。

步骤一：创建 GitHub 仓库

  1. 登录 github.com，点击右上角 + → New repository
  2. 仓库名建议：subnautica-2-guide（全小写，短横线分隔）
  3. 选择 Public（公开仓库，GitHub Pages 免费版必须公开）
  4. 点击 Create repository

步骤二：上传 HTML 文件

  1. 进入仓库页面，点击 Add file → Upload files
  2. 将本攻略的 index.html 文件拖入上传区域
  3. 填写提交信息（如 "Add Subnautica 2 guide"），点击 Commit changes

步骤三：开启 GitHub Pages

  1. 进入仓库 Settings（右上角齿轮图标）
  2. 左侧菜单找到 Pages
  3. Source 选 Deploy from a branch，Branch 选 main，目录选 / (root)
  4. 点击 Save，等待约 1–3 分钟
  5. 页面顶部会显示网址：https://你的用户名.github.io/subnautica-2-guide/
💡 加速被 Google 收录
部署完后去 Google Search Console 提交你的 URL。一般 1–7 天可以被收录。
